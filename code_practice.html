All about VIM

About Vim

Vim is a text editor that supports many coding languages and is available on many operating systems.

How to Setup your first file in the terminal

  1. type vim, then "file name"
  2. type "i" or "a" to go into insert mode
  3. type what you want
  4. type ":wq" to save and exit vim

Basic Commands

  • To open vim: vim "the name of the repo you want to make or open", and add "." and the coding language you want.
  • To exit vim: hit esc, then type :q
  • To save vim: hit esc, then type :w
  • To save and exit vim: hit esc, then type :wq
  • To edit: hit i, or a
  • To view syntax: hit esc, then type ":syntax on"
  • To delete: if you are in normal mode, type "x" to easily delete
